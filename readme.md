<h2>css : scroll-behavior</h2>

- scroll-behavior
  - 브라우저가 유저의 페이지 스크롤을 어떻게 동작시킬지 명시
  - value 타입
    - smooth : 부드러운 스크롤, 대부분 브라우저에서 default
    - auto( or instant) : 부드러운 효과 없이 즉시 스크롤

```
 element {
    scroll-behavior : value
 }

 value : `smooth` | `auto(instant)`
```
